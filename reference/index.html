API Reference¶

Filtrations¶

cechmate.filtrations.Alpha Construct an Alpha filtration from the given data.
cechmate.filtrations.Rips Construct a Rips filtration and the associated diagrams.
cechmate.filtrations.Cech Compute the Cech filtration of a Euclidean point set for simplices up to order self.max_dim.

Solvers¶

cechmate.solver.phat_diagrams Compute the persistence diagram for simplices using Phat.
